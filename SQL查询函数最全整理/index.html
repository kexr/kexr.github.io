SQL查询函数最全整理
发表于 2020-06-01| 更新于 2020-07-25

MYSQL

简单查询

  1. SELECT 列名1，列名2 FROM 表名;

    在数据表中查询以,分隔的所有列

  2. SELECT 列名 AS 别名，列名 AS 别名 FROM 表名;

    在数据表中查找列,为查找结果给定别名

  3. SELECT DISTINCT sex FROM emp;

    在表emp中查找sex列，结果做去重处理

  4. SELECT 运算式;

    可得出运算式结果

  5. SELECT * FROM 表名 ORDER BY 列名 DESC;

    根据BY后的列名进行排序降序

  6. SELECT * FROM 表名 ORDER BY 列名 ASC;

    根据BY后的列名进行升序降序

  7. SELECT 列名，列名 FROM 表名 WHERE 列名=列值;

    查找表中符合WHERE后条件的所有值

  8. SELECT 列名，列名 FROM 表名 WHERE 列名 IS NULL;

    查找条件列为空的值

  9. SELECT 列名，列名 FROM 表名 WHERE 列名 IS NOT NULL;

    查找条件列不为空的值


查询分句 条件说明
WHERE 列名 BETWEEN 5000 AND 7000; 条件列>5000并且<7000
WHERE 列名 BETWEEN 5000 AND 7000; 条件列在5000到7000之间的值
WHERE 列名 NOT BETWEEN 5000 AND 7000; 条件列不在5000到7000之间的值
WHERE 列名>7000 OR 列名<5000; 条件列>7000或者<5000
WHERE 列名=20 OR 列名=30; 条件列等于20或者30
WHERE 列名 IN(20,30); 条件列等于20或者30
WHERE 列名 NOT IN(20,30); 条件列不等于20并且不等于30
WHERE 列名 LIKE ‘%e%’; 模糊查询，其中%表示零个或者多个任意字符

分页查询

SELECT * FROM 表名 LIMIT 开始查询的值，每页数据量; 例每页5个值
SELECT * FROM 表名 LIMIT 0,5; 第一页
SELECT * FROM 表名 LIMIT 5,5; 第二页
SELECT * FROM 表名 LIMIT 10,5; 第三页

聚合查询

  1. SELECT MAX(列名) FROM

    查询最大值

  2. SELECT MIN(列名) FROM

    查询最小值

  3. SELECT COUNT(列名) FROM

    查询筛选所得列的条目总数

  4. SELECT SUM(列名) FROM

    查询筛选所得列的数据总和

  5. SELECT AVG(列名) FROM

    查询筛选所得列的数据平均值

  6. SELECT 列名 FROM 表名 GROUP BY 列名

    根据GROUP BY后的列为SELECT后要取出的列分组，常用于聚合函数

子查询

  1. SELECT * FROM 表名 WHERE 列名=(子查询语句)；

多表查询

  1. SELECT 列名 FROM 表名1 表名2 WHERE 列1=列2;
  2. SELECT 列 FROM 表1 INNER JOIN 表2 ON 列1=列2;

    内连接，当列1与列2中有一个值为空时即不显示该条记录

  3. SELECT 列 FROM 表1 LEFT OUTER JOIN 表2 ON 列1=列2;

    左外连接，当列1不为空，只有列2为空时，条目也可显示

  4. SELECT 列 FROM 表1 RIGHT OUTER JOIN 表2 ON 列1=列2;

    右外连接，列2不为空，列1可以为空

  5. (左外连接语句) UNION ALL (右外连接语句)

    显示全部符合要求的记录，其中如有列1，或者列2为空，也显示记录
